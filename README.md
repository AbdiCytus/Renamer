# Renamer

Program kecil untuk melakukan **rename** banyak sekaligus pada file gambar yang ada di dalam sebuah folder.

## Cara Menjalankan Program

- Clone repo `git clone https://github.com/AbdiCytus/Renamer.git`
- Masuk ke dalam direktori `cd Renamer`
- Instal semua depedency `npm install`
- Jalankan program `node index.js`
- Copy path foldernya kemudian paste ke dalam terminalnya
- Isikan nama baru untuk ke semua filenya
- Program akan mulai melakukan rename ke seluruh file gambarnya menjadi [nama-baru] - [n]
- Selesai
